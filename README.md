## Major Projects

### Zuvany — AI Customer Support & Sales Agent

Zuvany is a platform that enables businesses to deploy **AI agents on their websites** to automate customer interactions.

Capabilities:

* AI customer support agents
* lead capture and qualification
* automated workflows
* scalable chat infrastructure
* CRM integrations

Tech stack

Next.js • Node.js • Redis • MongoDB • AWS • Gemini

---

### AI CMS — AI Website CMS Generator

AI CMS is an **AI-driven CMS generation platform**.

Users can:

* describe their website idea
* paste an existing website URL

The AI automatically generates:

* CMS structure
* admin dashboard
* dynamic page layouts
* content collections

Users can also modify the design through **AI instructions**.

---

### Melhorme — Clinic Management System

Healthcare management platform designed for **multi-branch clinics**.

Roles:

* Patients
* Doctors
* Staff
* Admin

Features:

* appointment management
* patient records
* prescription handling
* clinic reporting
* role-based permissions

---

### Appoim — Professional Consulting Marketplace

Appoim is a **consulting marketplace platform similar to Topmate**.

Capabilities:

* expert profile listings
* availability scheduling
* appointment booking
* payment integration
* consultation management

### SME Ecommerce Platform
